# Stock-Market-Strategy-NIFTY-50-Backtesting-
Trinetra 2.0 is a trend-following, long-only trading strategy built in Pine Script v5 for TradingView. The idea behind this system is simple:  Use Heikin Ashi candles to reduce noise Combine dual EMAs (21 &amp; 55) to confirm market direction Deploy an ATR-based trailing stop to dynamically manage risk
